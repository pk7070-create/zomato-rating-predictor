# 🍔 Zomato Rating Predictor

A Machine Learning web application that predicts restaurant ratings based on user inputs like votes, cost, and restaurant type.

---

## 🚀 Project Overview

This project uses **Linear Regression** to predict restaurant ratings.  
It includes:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering (One-Hot Encoding)
- Model Training & Evaluation
- Web Deployment using Flask

---

## 🧠 Features Used

- Votes
- Cost for two people
- Online Order (Yes/No)
- Table Booking (Yes/No)
- Restaurant Type (Dining, Cafes, Buffet, Other)

---

## ⚙️ Tech Stack

- Python 🐍
- Pandas & NumPy
- Scikit-learn
- Flask (for deployment)
- HTML + CSS (UI)

---

## 📊 Model Details

- Algorithm: Linear Regression
- Evaluation Metrics:
  - MAE
  - MSE
  - R² Score
  - Cross Validation Score

---

## 🌐 Web App

The project includes a Flask-based web application where users can input restaurant details and get predicted ratings instantly.
